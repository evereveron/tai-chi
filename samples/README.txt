This readme provides the redistribution policy for the content in the Cisco Jabber SDK "samples" folder.

---------------------
REDISTRIBUTION POLICY
---------------------

Pursuant to the Cisco Jabber SDK License Agreement, the following content in the samples folder is redistributable when incorporated in a developer application:

- acd.html
- pinp.html
- popout.html
- sample.html
- style.css
- fullscreen.js
- sample_helper.js


For more information about the Cisco Jabber SDK, go to the Cisco Developer Network at https://developer.cisco.com/site/collaboration/jabber/websdk/overview/overview/
